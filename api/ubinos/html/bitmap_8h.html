◆ BITMAP_DEPTH_MAX

#define BITMAP_DEPTH_MAX   4

Max bit size is (INT_SIZE * 8) ^ BITMAP_DEPTH_MAX if BITMAP_DEPTH_MAX is 4 in 32 bit machine, then max bit size is 32 ^ = 1,048,576 bits
